Bài tập 1: Xây dựng trang web cá nhân của bạn

Mô tả: Hãy xây dựng một trang web CV cho bản thân bạn sau khi đã học xong khóa HTML và CSS của W3School.

Nội dung trang web gồm có:

  • Trang chủ chứa những thông tin cơ bản về bản thân.
  • Các trang chi tiết cung cấp các thông tin cụ thể về học vấn, kỹ năng, v.v...

Mục tiêu: Xây dựng được trang web với giao diện như thế này: Website. Tuy nhiên ta cần xây dựng nội dung khung sườn cho trang web trước và đạt tối thiểu như thế này: Website

Yêu cầu:

  • Sau khi học xong khóa HTML của W3School, hãy dựng trang HTML giống như thế này: Website
  • Sau khi học xong khóa học CSS trên W3School, hãy kết hợp các trang web vừa làm với CSS để có được một giao diện như thế này: Website

Quy định làm bài:

  • Giao diện phải được layout phù hợp
  • Không yêu cầu phải responsive
  • Không được dùng CSS Framework.
  • Không được nhờ bất kỳ ai hướng dẫn, phải tự tìm hiểu và thực hiện.
  • Không được xem trước source code.
