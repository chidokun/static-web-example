Bài tập 1: Xây dựng trang web cá nhân của bạn

Mô tả: Hãy xây dựng một trang web CV cho bản thân bạn sau khi đã học xong khóa HTML và CSS trên W3Schools.

Nội dung trang web gồm có:

  • Trang chủ chứa những thông tin cơ bản về bản thân.
  • Các trang chi tiết cung cấp các thông tin cụ thể về học vấn, kỹ năng, v.v...

Mục tiêu: Xây dựng được trang web với giao diện như thế này: Website. Tuy nhiên ta cần xây dựng nội dung khung sườn cho trang web trước và đạt tối thiểu như thế này: Website

Yêu cầu:

  • Sau khi học xong khóa HTML của W3School, hãy dựng trang HTML giống như thế này: Website
  • Sau khi học xong khóa học CSS trên W3Schools, hãy kết hợp các trang web vừa làm với CSS để có được một giao diện như thế này: Website

Quy định làm bài:

  • Giao diện phải được layout phù hợp
  • Không yêu cầu phải responsive
  • Không được dùng CSS Framework.
  • Không được nhờ bất kỳ ai hướng dẫn, phải tự tìm hiểu và thực hiện.
  • Không được xem trước source code.

Bài tập 2: HTML + CSS + JavaScript

Mô tả: Sau khi học thêm khóa JavaScript trên W3Schools, hãy xây dựng một ứng dụng Web đơn giản cho phép người dùng chọn các chỉ số RGB để lấy mã màu.

Mục tiêu: Xây dựng được ứng dụng đơn giản bằng JavaScript với tất cả các tính năng như ứng dụng mẫu này: Website

Tính năng gồm có:

  • Các thanh trượt cho phép người dùng chọn các giá trị màu RGB.
  • Khung thể hiện màu tổng hợp được chọn và mã màu của nó.
  • Xem thông tin tác giả viết ứng dụng.

Tỉ lệ kiến thức cần để thực hiện: HTML (20%), CSS (70%), JavaScript (10%).

Các bước làm:

  1. Nhìn ứng dụng mẫu, suy nghĩ hướng bố trí thẻ, phân vùng sao cho dễ dàng layout nhất.
  2. Suy nghĩ cách lấy giá trị màu, hiển thị màu, thuật toán tìm màu.
  3. Dựng trang HTML theo hướng bố trí đã suy nghĩ.
  4. Dựng CSS để hoàn thiện giao diện.
  5. Xử lý JavaScript theo thuật toán đã suy nghĩ.
  6. Hiệu chỉnh lại giao diện, bố trí.

Quy định làm bài:

  • Giao diện phải giống như ứng dụng mẫu và phải đẹp.
  • Không yêu cầu phải responsive.
  • Không được dùng CSS Framework.
  • Không được nhờ bất kỳ ai hướng dẫn, phải tự tìm hiểu và thực hiện.
  • Không được xem trước source code.
